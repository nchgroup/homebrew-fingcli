# homebrew-fing
homebrew format of nchgroup/fingcli

## Installation
```bash
$ brew tap nchgroup/fingcli
$ brew install fingcli
```

### Direct installation
```bash
$ brew install nchgroup/fingcli/fingcli
```

## Remove hombrebrew
```bash
$ brew untap nchgroup/fingcli
```

## Update
```bash
$ brew update
$ brew upgrade fingcli
```
